# TastyDucks' Homebrew Tap

Install formulae from this tap directly:

```bash
brew install tastyducks/tap/nebius
```

Or tap the repository first:

```bash
brew tap tastyducks/tap
brew install nebius
```

## HIDAPITester

Simply installs [version 0.5](https://github.com/todbot/hidapitester/releases/tag/v0.5).

## Nebius CLI

The `nebius` formula installs the Nebius AI Cloud CLI.

Updates are checked hourly from the Nebius CLI Mintlify RSS feed. When a new version appears, CI updates the formula SHA256 values, installs and tests the formula, then opens a pull request.
